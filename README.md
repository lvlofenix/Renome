Renome de arquivos em massa

Software para organização de nomes em arquivos.

# Versão 1.0.1
 Lingua: Portugues e Ingles (Google Translate) 
# Opções
> Por Ordem numérica.

> Nome fixo

> Alfanumérico 

> Tamanho

> Data de criação

> Tipo de arquivo

> Forma Aleatoria por numéro, Letras e Alfanuméricos.
